# Contributing

Contributions are welcome! We value feedback and contributions from the community.

## How to Contribute

- **Reporting Bugs**: If you find a bug, please open an issue and provide as much detail as possible, including your operating system, package versions, and steps to reproduce the issue.
- **Suggesting Enhancements**: If you have an idea for an enhancement, please open an issue to discuss it.
- **Pull Requests**: We welcome pull requests. Please ensure your code follows the project's style and that all tests pass.

## Development Setup

1.  Clone the repository and `cd` into the directory.
2.  Create and activate a virtual environment.
3.  Install dependencies with `pip install -r requirements.txt`.

Thank you for contributing!
