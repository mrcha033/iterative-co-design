# Contrib/Style & ADR Template

One-line: Define coding style, review rules, and ADR template.

## Style
- Python: black+ruff, type hints, no one-letter names.
- Tests: pytest, minimal external deps.

## Reviews
- Small PRs, include `Acceptance` checklist and links to docs.

## ADR Template
```
Title:
Date:
Context:
Decision:
Consequences:
Alternatives:
Linked Docs/PRs:
```
